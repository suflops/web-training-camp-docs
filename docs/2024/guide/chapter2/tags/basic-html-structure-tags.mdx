# HTML基本结构标签介绍

我们来解决在HTML的基本结构那一节的遗留问题：HTML基本结构标签的介绍

```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body></body>
</html>
```

:::danger{title="重要"}
`<!DOCTYPE>` **不是一个 HTML 标签**，而是一个文档类型声明（Document Type Declaration，简称 DTD）。

它用于告知浏览器应该使用哪种标准来解析和渲染 HTML 文档。虽然它在 HTML 文档的开头位置出现，看起来像一个标签，但它实际上是一个指令，具有特殊的语法和用途。
:::

## 一、`<html>` 标签

`<html>` 标签是 HTML 文档的根元素，它包含了整个 HTML 文档的内容，是所有 HTML 元素的容器。

其中 `lang` 属性用于指定文档的语言，以便浏览器正确地显示文本内容。

目前 `lang` 属性的值有很多种，常见的有：

- `en`：英语
- `zh`：中文

等等，当使用中文浏览器时，如果`lang` 属性设置为 `en` 时，浏览器会跳出是否翻译的提示，如果设置为 `zh` 时，浏览器将不会提示是否翻译，同学们在使用浏览器的时候可以尝试一下。

## 二、`<head>` 标签

`<head>` 标签用于定义文档的头部，它包含了文档的元数据（metadata），如文档的标题、字符编码、视口设置等。

:::info{title="元数据："}

文档元数据（Document Metadata）是指提供关于 HTML 文档本身的信息的数据。元数据并不是网页内容的一部分，但它为浏览器、搜索引擎和其他服务提供了关于页面的有用信息。

列如：`<title>` 这个标签就是用来定义文档的标题，它并不会直接展示在页面主体中。

:::

## 三、`<meta>` 标签

`<meta>` 标签用于设置 HTML 文档的元数据，它是一个空元素，不需要结束标签。

`<meta>` 标签通常用于指定字符编码、视口设置、关键字、描述等信息。

- `name`：用于指定元数据的名称。
- `content`：用于指定元数据的内容。

列如：

- `<meta charset="UTF-8">` 用于指定文档的字符编码为 UTF-8。
- `<meta name="viewport" content="width=device-width, initial-scale=1.0">` 用于设置视口的宽度等于设备的宽度，并且初始缩放比例为 1.0。

### 其他`meta`元数据标签

```html
<meta name="description" content="The description about this web">
```
- 作用：提供网页的简短描述。
- 意义：
    - 搜索引擎会使用这个描述在搜索结果页面中显示页面的摘要。这可以帮助用户了解页面内容，并决定是否点击链接。
    - 对于社交媒体分享，也可能用这个描述作为预览文本。

```html
<meta name="keywords" content="keywords">
```

- 作用：提供与页面内容相关的关键词。
- 意义：
    - 早期的搜索引擎使用这些关键词来帮助确定页面的主题和相关性。然而，现代搜索引擎大多不再依赖 keywords 元数据，因为它容易被滥用。

```html
<meta name="author" content="author">
``` 
- 作用：提供页面或站点的作者信息。

- 意义：
    - 表明谁是页面的创建者或所有者。对于多个作者或团队项目，可以提高透明度和责任感。有助于内容管理和版本控制。

## 四、`<title>`

`<title>` 标签用于定义文档的标题, 如果没有设置 `<title>` 标签，浏览器会使用文件名作为标题。

## 五、`<body>` 标签

`<body>` 标签用于定义文档的主体，它包含了文档的内容，如文本、图片、链接等，是用户看到的页面内容的展示区域。


## 本节涉及到的标签

- `<html>`：定义 HTML 文档的根元素。[双标签]
- `<head>`：定义文档的头部。[双标签]
- `<meta>`：定义文档的元数据。[单标签]
- `<title>`：定义文档的标题。[双标签]
- `<body>`：定义文档的主体。[双标签]
